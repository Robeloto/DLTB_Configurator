DLTB Configurator by Robeloto
version 0.6.0-beta

GUI tool to build & install a custom PAK for Dying Light: The Beast.

Download:
Go to releases and download the latest version.

What's new in version 0.60b?

• Changed UI theme to a darker theme
• Added a top toolbar for preset/save-path/game-folder actions
• Added small icons to the tab headers
• Mods tab for loading 3rd party mods
• Separated Main and XP section
• Jumping height added
• Movement speed up to 300%

Quick start:
Run DLTB Configurator.exe
Set Game Folder
(Optional) Set Save Folder (backup)
Apply → Build & Install PAK

Features:
XP multipliers (Open World + Legend)
Flashlight presets (drain/regen/colors)
Enemies - tweak health - chase limit sliders
Hunger + restore
Volatile tuning + health multipliers
Vehicle keybinds + health and fuel sliders
Auto-detect game folder + save-backup
Mods - Open Nexus Mods - Download zip/rar - Install - Deploy - Apply
One-click Build & Install PAK
(if third party mods installed - Run supermodmerger)
Play game from the GUI or via STEAM.

Notes:
Hunger restore to 100% does not work.
Spawns disabled since v1.5+

Troubleshooting:
“Multiple Steam saves found” → manual path
Some keybindings might not work for Vehicle
